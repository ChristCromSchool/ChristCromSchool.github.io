Project: CS-465 - Web Application Enhancements

This project showcases several enhancements made to a web application, focusing on various aspects such as testing improvements, Google authentication, pagination, search functionality, and Stripe integration for payment processing. This enhancement focuses on improvements to testing, ensuring the application is more robust and reliable. The testing improvements were designed to increase the efficiency and accuracy of the system.

This update introduces Google authentication for better security, along with pagination and a search bar to enhance the user interface and experience. These features make the application more user-friendly and efficient.

This enhancement focuses on integrating a shopping cart and Stripe for secure payment processing. This is a critical feature for e-commerce applications, allowing users to complete transactions seamlessly.
